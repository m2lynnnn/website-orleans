# Orleans FA - Site Web

Site officiel contenant le règlement du serveur Discord Orleans FA.

## Installation

### Prérequis
- Node.js 14+ installé
- npm ou yarn

### Étapes

```bash
# 1. Cloner ou télécharger le projet
cd website-orleans

# 2. Installer les dépendances
npm install

# 3. Lancer le serveur
npm start

# OU en développement (avec auto-reload)
npm run dev
